## Express API with Typescript, MongoDB, Mongoose, Jest, Swagger and Docker
[]: # 
[]: # ## How to run
[]: # 
[]: # ### Local
[]: # 
[]: # 1. Install dependencies
[]: # ```bash
[]: # npm install
[]: # ```
[]: # 2. Start the server
[]: # ```bash
[]: # npm run dev
[]: # ```
[]: # 
[]: # ### Docker
[]: # 
[]: # 1. Build the image
[]: # ```bash
[]: # docker build -t express-api .
[]: # ```
[]: # 2. Run the container
[]: # ```bash
[]: # docker run -p 3000:3000 express-api
[]: # ```
